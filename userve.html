    uServe

    An offline server. Scroll to read more

    Information at the go | Affordable | Relevant Content | Digital Safety and Security | Reliable Power Supply

    Digital community Library Home use for kids Computer Labs of Professionals (nurses, teachers)

    for school high schools in rural areas


    UServe is a microserver technology which provides internet access service delivered in an offline mode with customized content to suit any client from any field based on their needs. Users only have to search from a common search engine like any normal browser and results from all the sources will be displayed for user to select any option of their choose. It provides both a wireless option and an Ethernet option with a wireless adaptor and switch respectively which can serve about fifteen (15) computers at a time

    Content

    The content of the server is gathered from three main sources:

    1. Relevant online website which have offline versions like Wikepedia and Khan Academy.

    2. Self-generated content from the UServe Company which is tailored to meet the needs of any category of clients/users.

    3. Content provided by client.

    Update

    Due to the fact that information is not static, there is the need for a regular update of the system. Userve will is updated both manually and online every 3 months.

    Hardware

    Raspberry Pi B+

    Battery (5v, 12000mAh)

    Ethernet Switch (optional)

    External Hard Disk Drive Storage

    Control circuitry

    USB wifi dongle

    LEDS

    Software

    The server (LLMP) is installed on the Raspberry Pi running Raspbian OS (wheezy stable). Dynamic Host Configuration Protocol (DHCP) Server is also installed on the Raspberry Pi to enable proper IP address allocation on the local network. The external hard drive is used to store the materials which the server will be sharing. Similar to browsing the web, Userve is accessed by opening the homepage of a browser and typing the Userve address in the address bar. On the homepage, a search bar helps the user to look for content like any normal browser. When a desired link is clicked, the index file of that web folder is accessed by the web browser and shows the homepage of the desired website. If the link is more specific however, the specific page in the folder will be shown instead. Users can also for services like ‘online’ quizzes and exams which keep track of their progress. The user’s details as well as scores are stored in an SQL database through PHP scripts and are cross-checked each time the user logs in.

    Why Should You Get Userve/Advantages

    1. It has the huge advantage of protecting children, students and the general public from unwanted contents on the internet thereby giving them a level of concentration on their studies.

    2. It does not require any subscription to any data package, thus its ‘virtually free’ after purchase.

    3. It has a lot of local content; making it very easy to use and understand by the local user (any country around the world).

    4. It has a back-up power storage that enables it to stay on for 2-8 hours off the grid.

    5. Its portable and appealing packing makes it easy to carry around and use in any environment.
